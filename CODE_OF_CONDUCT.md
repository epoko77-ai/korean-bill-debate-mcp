# Code of Conduct

We are committed to a welcoming, harassment-free project. Be respectful, discuss evidence and
technical tradeoffs rather than people, and assume good faith. Harassment, discrimination,
threats, doxxing, and sustained disruption are not accepted.

Project maintainers may edit or remove harmful contributions and restrict participation when
needed. Report conduct concerns privately through the repository owner's published GitHub
contact channel. Maintainers will protect reporter privacy as far as practical.
